文档

图形定位器，用来动态的决定某个图形相对于另外一个图形的位置

一般来讲，如果你希望一个图形 能跟着另外一个图形移动，那么Locator就很有用了。

如：Connection Label（连线上的文字）

MidpointLocator:定位到连线中点

MidpointOffsetLocator：连线中点并且可以设置偏移量
